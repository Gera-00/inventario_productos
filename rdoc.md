# Ejercicio Inventario de productos

    1. Descripción: Control de existencias de productos con nombre, categoría, cantidad y precio.
        
    2. Consultas: Alerta visual cuando un producto esté por agotarse.
        
    3. Tablas: productos, categorías.
        
        °Productos{id, nombre, id_categoria, cantidad/stock, precio}

        °Categorias{id, nombre}
        
        °Usuarios{id, nombre, usuario, contrasenia}

        
    4. Diseño obligatorio: Bootstrap y/o  CSS para diseño.
        
    5. Validaciones : JavaScript.
        
    6. Empleo de patrones de diseño, Responsivo
        
    7. Autentificación: php
